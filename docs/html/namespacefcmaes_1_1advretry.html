Retries a loop for optimization until stopping criteria are met.

The function performs optimization processes in a loop, sampling solutions and evaluating their fitness to improve a defined objective within bounds. It also incorporates handling parallel executions for optimization using private random generator and thread limits.

Retries the optimization process using multiple worker processes and random number generators.

This function parallelizes the optimization task, applies a stopping criterion based on the provided fitness value, and selects the best result from the optimization attempts.

Parameters
storeStores results of optimization during the process.
optimizeThe optimization function to be applied.
value_limitThe upper limit for the function value considered in optimization. Defaults to positive infinity (np.inf).
workersThe number of parallel workers to allocate for the process. Defaults to the total number of CPU cores available.
stop_fitnessThe stopping fitness criterion for the optimization process. The optimization stops when this value is reached. Defaults to negative infinity (-np.inf).
Returns
OptimizeResult The result of the optimization process containing the best solution found (x), the fitness of this solution (fun), the number of function evaluations (nfev), and a success flag (success).
