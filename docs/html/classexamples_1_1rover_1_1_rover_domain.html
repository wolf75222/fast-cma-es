Detailed Description

Rover domain defined on R^d cost_fn vectorized function giving a scalar cost to states start a start state for the rover goal a goal state traj a parameterized trajectory object offering an interface to interpolate point on the trajectory s_range the min and max of the state with s_range[0] in R^d are the mins and s_range[1] in R^d are the maxs.
