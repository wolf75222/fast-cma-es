Detailed Description

A wrapper class to manage function calls with additional features such as tracking progress, logging results, and storing or plotting outcomes.

This class allows function evaluation with multiprocess support, tracks the number of evaluations, records the best observed outcomes, logs key metrics at specified intervals, and interfaces with optional storage or plotting mechanisms.

fun (Callable[[ArrayLike], ArrayLike]): The function to be wrapped and called.
