Detailed Description

Represents a fitness evaluation system for optimization problems.

This class encapsulates a fitness evaluation system that supports parallel evaluation, manages scaling and normalization of input values based on given bounds, and provides encoding and decoding mechanisms for the input values. It tracks the number of fitness evaluations performed and ensures that inputs conform to specified feasible bounds.
