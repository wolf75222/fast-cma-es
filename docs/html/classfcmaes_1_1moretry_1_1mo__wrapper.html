Detailed Description

Wraps a multi-objective evaluation function with additional functionality.

This class encapsulates a multi-objective evaluation function, integrating weights for objectives and constraints, and provides methods to evaluate the function with weight-driven adjustments and constraints handling. It is intended for use in scenarios requiring objective evaluations with flexible weighting and constraint violations.
