1"""Find the optimal subset fulfilling any property.
2Multi-objective variant of https://github.com/dietmarwo/fast-cma-es/blob/master/examples/subset.py
3
4As example we use transactions where a subset is to be matched to a list of payments.
5For transactions: [2,4,5,1,3] and payments: [4, 4] the sum of payments is 8.
6Subsets of transactions that minimize the difference between payments and
7transactions are for instance [5,3], or [4,3,1]. As second objective we maximize
8the minimal transaction in the chosen subset.
9
10In general we have to define a mapping 'selection_value' which maps a
11specific selection/subset represented as boolean array to a list of values to be minimized.
12
13See https://github.com/dietmarwo/fast-cma-es/blob/master/tutorials/Subset.adoc for a detailed description.
14"""

30class transaction_value():
31
32 def __init__(self, transactions, payments):
33 self.transactions = transactions
34 self.sum_payments = sum(payments)
35
36 def __call__(self, selection):
37 trs = self.transactions[selection]
38 return abs(sum(trs) - self.sum_payments), -min(trs)
