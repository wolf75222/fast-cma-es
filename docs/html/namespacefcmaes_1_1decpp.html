Performs optimization using the Differential Evolution algorithm.

This function seeks to find the global minimum of a given objective function by employing the Differential Evolution strategy. It supports various configurable parameters, including bounds, population size, mutation factors, and customization of the optimization process through callbacks and other options.

   fun (Callable[[ArrayLike], float]): The objective function to minimize.
       Must take an array-like input and return a scalar float value.
Parameters

boundsBounds on the input parameters. Should be provided as a sequence of tuples (min, max) or equivalent.
popsizeSize of the population for evolution. Defaults to 31.
max_evaluationsMaximum number of function evaluations allowed. Defaults to 100000.
stop_fitnessFitness threshold at which optimization stops. Defaults to -infinity.
