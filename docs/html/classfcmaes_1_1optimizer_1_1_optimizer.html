Detailed Description

Provides functionalities for optimization tasks.

This class is designed to handle optimization operations, allowing users to set a maximum number of evaluations and assign a specific name to the optimizer. It also has methods for retrieving the maximum evaluation number and counting runs, optionally integrating with a store object.

   max_evaluations (int, optional): Maximum number of evaluations allowed for
       the optimizer. Default is 50000.
   name (str, optional): The name assigned to the optimizer. Default is an
       empty string.
