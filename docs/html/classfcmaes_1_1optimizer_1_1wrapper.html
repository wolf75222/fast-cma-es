Detailed Description

Wrapper for optimization evaluation and logging.

This class serves as a wrapper around a user-defined objective function (fit). It tracks the number of function evaluations, the best evaluation value found so far, and provides logging functionality. The wrapper is typically used in optimization tasks to facilitate performance tracking and debug information.

   fit (Callable[[ArrayLike], float]): The objective function to be
       instantiated. Used for tracking elapsed time.
